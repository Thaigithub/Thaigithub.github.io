Term-Structure Extraction

Bootstrapping and Valuation of Swap

Extract an interest-rate term structure from market data and use it to value swaps with transparency and rigour.

Fixed Income Term Structure Bootstrapping Python

Overview

  • Goal: derive a term structure (zero curve / discount factors) and price swaps consistently.
  • Approach: bootstrap from market instruments interpolate yields/discounts compute present values of cash flows.

Quick reference

  • Instruments: swaps, bills, coupon bonds
  • Interpolation: log-linear

Term structure construction

Workflow

  • Bootstrap short-end (BBSW) using money-market conventions.
  • Solve swap PV equations to infer mid/long-term discount factors.
  • Log-linear Interpolate to fill the curve between pillars.
  • Convert: discount factor term structure $B(0,t)$ spot instantaneous forward rate $f(0,t)$

Outputs

  • discount factors $B(0,t)$, and interpolated term-structure.

Assumption

  • Day count convention is 30/360

Rate formulas $$ \begin{align*} \text{BBSW}(T) &= \frac{1}{T} \left( \frac{1}{B(0,T)} - 1 \right) \\ \text{Swap rate}(0,\mathbb{T}) &= \frac{1 - B(0,T_n)}{\sum_{i=1}^{N} (T_i - T_{i-1}) B(0,T_i)} \\ \end{align*} $$ Log-linear interpolation of discount factors $$ \log B(0,t) = \log B(0,t_i) + \frac{t - t_i}{t_{i+1} - t_i} \left(\log B(0,t_{i+1}) - \log B(0,t_i)\right), \quad t \in [t_i, t_{i+1}] $$

def term_structure_constructor(df):
    new_df = pd.DataFrame(df)
    ts = np.array(df["Term"].str.split(" ", expand=True)[0].astype(float).values)
    ts[:6] = ts[:6] / 12
    new_df["Term"] = ts
    new_df["Rate"] = new_df["Rate"] / 100

    def target_func(x):
        B = np.array([1] + list(x))
        B = np.log(B)
        T = np.array([0] + list(new_df["Term"]))
        interp = interp1d(T, B, kind="linear", fill_value="interpolate")

        def log_lin(t):
            return np.exp(interp(t))

        equations = []
        for i in range(len(new_df)):
            if new_df.iloc[i, 0] == "BBSW":
                equations.append(
                    (1 / log_lin(new_df.iloc[i, 1]) - 1) / new_df.iloc[i, 1]
                    - new_df.iloc[i, 2]
                )
            else:
                T_i = new_df.iloc[i, 1]
                rate = new_df.iloc[i, 2]
                freq = 1 / 4 if T_i <= 3 else 1 / 2
                payments = np.arange(freq, T_i + freq, freq)
                sum_dfs = np.sum([log_lin(t) for t in payments])
                equations.append((1 - log_lin(T_i)) / (freq * sum_dfs) - rate)
        return equations

    B = fsolve(target_func, [0.5] * len(new_df))
    B = np.array([1] + list(B))
    B = np.log(B)
    T = np.array([0] + list(new_df["Term"]))
    interp = interp1d(T, B, kind="linear", fill_value="interpolate")

    def log_lin(t):
        return np.exp(interp(t))

    # prepend a zero-term row (time 0, DF = 1 implicitly via interpolation)
    new_row = pd.DataFrame([{"Type": "Cash", "Term": 0.0, "Rate": 0.0}])
    new_df = pd.concat([new_row, new_df], ignore_index=True)
    new_df["Discount Factor"] = [log_lin(t) for t in new_df["Term"]]

    def ifr_calculator(t):
        idx_start = new_df[new_df["Term"] <= t].index[-1]
        if idx_start == len(new_df) - 1:
            idx_start -= 1
        T1 = new_df.loc[idx_start, "Term"]
        T2 = new_df.loc[idx_start + 1, "Term"]
        B1 = np.log(new_df.loc[idx_start, "Discount Factor"])
        B2 = np.log(new_df.loc[idx_start + 1, "Discount Factor"])
        slope = (B2 - B1) / (T2 - T1)
        return -slope

    return log_lin, new_df, ifr_calculator

Term structure of three-month simple compounded forward rates

Term structure of three-month simple compounded forward rates implied by the term structure of discount factors can be plotted as follows: $$ \begin{align*} \text{r}(t,T,T+\delta) &= \frac{1}{\delta} \left( \frac{B(t,T)}{B(t,T+\delta)} - 1 \right) \\ \end{align*} $$

def simple_compounding_forward_rate_calculator(t):
    if t < 0 or t > term_structure.iloc[-1, 1] - 1 / 4:
        return ValueError(f"t must be in range [0,{term_structure.iloc[-1, 1]-1/4}]")
    B_T1 = df_calculator(t)
    B_T2 = df_calculator(t + 1 / 4)
    return (B_T1 / B_T2 - 1) * 4

Swap valuation

Swap rate calculation

Consider an eleven-year interest rate swap paying fixed and receiving floating semi-annually. Determine the fixed rate of this swap such that the value of the swap is zero.

$$ \begin{align*} \text{Swap rate}(0,\mathbb{T}) &= \frac{1 - B(0,T_n)}{\sum_{i=1}^{22} (T_i - T_{i-1}) B(0,T_i)} \\ T_i &= \frac{i}{2}, \quad i = 1, 2, \ldots, 22 \\ \end{align*} $$

Swap valuation

Suppose that one month ago you entered into a two-year interest rate swap paying a fixed rate of 3.7% and receiving floating (based on three-month BBSW) quarterly. Three-month BBSW on 30 June 2025 was 3.6037%.

$$ \begin{align*} V &= V_{\text{floating}} - V_{\text{fixed}} \\ V_{\text{fixed}} &= \ell(\mathbb{T}) \times \sum_{i=1}^{8} (T_i - T_{i-1}) B(0,T_i) \\ V_{\text{floating}} &= r(T_0,T_1)(T_1-T_0)B(0,T_1)+ (1 - B(T_1,T_8))B(0,T_1) \\ T_i &= \frac{i}{2}-\frac{1}{12}, \quad i = 0, 1, 2, \ldots, 8 \\ \end{align*} $$
