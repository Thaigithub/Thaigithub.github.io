Pricing Coupon Bonds via Term-Structure Extraction

Extract an interest-rate term structure from market data and use it to value coupon bonds with transparency and rigour.

Fixed Income Yield Curve Bootstrapping Python

Overview

  • Goal: derive a term structure (zero curve / discount factors) and price coupon bonds consistently.
  • Approach: bootstrap from market instruments interpolate yields/discounts compute present values of cash flows.

Quick reference

  • Instruments: swaps, bills, coupon bonds
  • Interpolation: log-linear / maximum smoothness

Term structure construction

Boostrapping and log-linear interpolation

Workflow

  • Bootstrap short-end (BBSW) using money-market conventions.
  • Solve swap PV equations to infer mid/long-term discount factors.
  • Log-linear Interpolate to fill the curve between pillars.
  • Convert: discount factor term structure $B(0,t)$ spot instantaneous forward rate $f(0,t)$

Outputs

  • discount factors D(t), and interpolated term-structure.

Tip: clearly state the compounding and day-count conventions used when publishing the curve.

$$ \begin{align*} \text{BBSW}(T) &= \frac{1}{T} \left( \frac{1}{B(0,T)} - 1 \right) \\ \text{Swap rate}(0,\mathbb{T}) &= \frac{1 - B(0,T_n)}{\sum_{i=1}^{N} (T_i - T_{i-1}) B(0,T_i)} \\ \end{align*} $$

Log-linear interpolation of discount factors: $$ \log B(0,t) = \log B(0,t_i) + \frac{t - t_i}{t_{i+1} - t_i} \left(\log B(0,t_{i+1}) - \log B(0,t_i)\right), \quad t \in [t_i, t_{i+1}] $$

import re


def term_structure_constructor(df):
    new_df = pd.DataFrame(df)
    new_df["Term"] = [1 / 12, 1 / 4, 1 / 2, 1, 3, 5, 10, 15]
    new_df["Rate"] = new_df["Rate"] / 100

    def target_func(x):
        B = np.array([1] + list(x))
        B = np.log(B)
        T = np.array([0] + list(new_df["Term"]))
        interp = interp1d(T, B, kind="linear", fill_value="interpolate")

        def log_lin(t):
            return np.exp(interp(t))

        equations = []
        for i in range(len(new_df)):
            if new_df.iloc[i, 0] == "BBSW":
                equations.append(
                    (1 / log_lin(new_df.iloc[i, 1]) - 1) / new_df.iloc[i, 1]
                    - new_df.iloc[i, 2]
                )
            else:
                T_i = new_df.iloc[i, 1]
                rate = new_df.iloc[i, 2]
                freq = 1 / 4 if T_i <= 3 else 1 / 2
                payments = np.arange(freq, T_i + freq, freq)
                sum_dfs = np.sum([log_lin(t) for t in payments])
                equations.append((1 - log_lin(T_i)) / (freq * sum_dfs) - rate)
        return equations

    B = fsolve(target_func, [0.95] * len(new_df))
    B = np.array([1] + list(B))
    B = np.log(B)
    T = np.array([0] + list(new_df["Term"]))
    interp = interp1d(T, B, kind="linear", fill_value="interpolate")

    def log_lin(t):
        return np.exp(interp(t))

    # prepend a zero-term row (time 0, DF = 1 implicitly via interpolation)
    new_row = pd.DataFrame([{"Type": "Cash", "Term": 0.0, "Rate": 0.0}])
    new_df = pd.concat([new_row, new_df], ignore_index=True)
    new_df["Discount Factor"] = [log_lin(t) for t in new_df["Term"]]

    def ifr_calculator(t):
        idx_start = new_df[new_df["Term"] <= t].index[-1]
        if idx_start == len(new_df) - 1:
            idx_start -= 1
        T1 = new_df.loc[idx_start, "Term"]
        T2 = new_df.loc[idx_start + 1, "Term"]
        B1 = np.log(new_df.loc[idx_start, "Discount Factor"])
        B2 = np.log(new_df.loc[idx_start + 1, "Discount Factor"])
        slope = (B2 - B1) / (T2 - T1)
        return -slope

    return log_lin, new_df, ifr_calculator

Maximum smoothness

Constructing the Term Structure via Maximum Smoothness

The maximum-smoothness approach constructs an instantaneous forward-rate curve $f(0,t)$ that both fits market instruments and minimises unnecessary curvature. This ensures a stable, arbitrage-free, and visually smooth term structure.

Smoothness Objective

We select the forward curve that minimises the integral of squared curvature:

$$ \min_{f} \int_{0}^{T} \left(f''(0,s)\right)^2\,ds. $$

This penalises rapid oscillations and ensures the smoothest curve consistent with market data.

Forward Curve Representation

Let the market provide discount factors at knot points

$$ 0 = t_0 < t_1 < \dots < t_m < t_{m+1} = T. $$

The maximum-smoothness solution (Vasicek spline) implies that on each interval:

$$ f(0,t) = c_i t^4 + b_i t + a_i, \qquad t_{i-1} < t \le t_i. $$

This is a piecewise quartic polynomial with no quadratic or cubic terms, resulting from the EulerLagrange solution of the variational problem.

Constraints

To ensure global smoothness and exact market fit, the coefficients ((a_i,b_i,c_i)) satisfy:

1. Continuity of the forward rate

$$ c_i t_i^4 + b_i t_i + a_i = c_{i+1} t_i^4 + b_{i+1} t_i + a_{i+1}. $$

2. Continuity of the slope

$$ 4c_i t_i^3 + b_i = 4c_{i+1} t_i^3 + b_{i+1}. $$

3. Exact fit to discount factors

For each maturity $t_i$, $$ \int_{t_{i-1}}^{t_i} f(0,s)\,ds = -\log \frac{B(0,t_i)}{B(0,t_{i-1})}. $$

Expanding the integral gives: $$ \frac{1}{5}c_i(t_i^5 - t_{i-1}^5) + \frac{1}{2}b_i(t_i^2 - t_{i-1}^2) + a_i (t_i - t_{i-1}) = -\log \frac{B(0,t_i)}{B(0,t_{i-1})}. $$

4. Terminal smoothness

$$ c_{m+1} = 0. $$

Common additional conditions:

  • Flat terminal slope: $ f'(0,T) = 0 \Rightarrow b_{m+1} = 0. $
  • Known short rate: $ a_1 = r(0). $

These yield a fully determined linear system for all coefficients.

Using Swap Rates Instead of Discount Factors

If the inputs are swap rates $ \ell(0,t_i) $ rather than zero-coupon prices, the pricing constraint becomes nonlinear:

$$ 1 = e^{-\int_0^{t_i} f(0,s)\,ds} + \ell(0,t_i)\sum_{j=1}^{i} e^{-\int_0^{t_j} f(0,s)\,ds}. $$

This transforms the calibration into a fixed-point problem, but the smoothness and continuity constraints remain identical.

Summary

  • The maximum-smoothness approach produces the smoothest instantaneous forward curve consistent with observed prices.
  • The optimal curve is a piecewise quartic spline.
  • Forward-rate continuity, slope continuity, and pricing constraints uniquely determine the spline coefficients.
  • Swap-rate inputs require a nonlinear solver; discount factors yield a linear one.

This method is widely used in fixed-income modelling due to its stability and arbitrage-free construction.

def smoothness_constructor(df, term_structure):
    new_df = pd.DataFrame(df)
    new_df["Term"] = [1 / 12, 1 / 4, 1 / 2, 1, 3, 5, 10, 15]
    new_df["Rate"] = new_df["Rate"] / 100
    points = len(new_df)
    BBSW = new_df[new_df["Type"] == "BBSW"].reset_index(drop=True)
    Swap = new_df[new_df["Type"] == "Swap"].reset_index(drop=True)

    def solve(p):
        results = []
        a = p[:points]
        b = p[points : 2 * points]
        c = p[2 * points :]

        # Condition 1
        for i in range(points - 1):
            t = new_df.iloc[i, 1]
            results.append(
                (c[i] * (t**4) + b[i] * t + a[i])
                - (c[i + 1] * (t**4) + b[i + 1] * t + a[i + 1])
            )

        # Condition 2
        for i in range(points - 1):
            t = new_df.iloc[i, 1]
            results.append(
                (4 * c[i] * (t**3) + b[i]) - (4 * c[i + 1] * (t**3) + b[i + 1])
            )

        # Condition 3
        results.append(b[-1])
        results.append(a[0] - new_df.iloc[0, 2])

        # Condition 4
        for i in range(len(BBSW)):
            ti = BBSW.iloc[i, 1]
            t = BBSW.iloc[i - 1, 1] if i != 0 else 0
            B_prev = (
                1 if i == 0 else (1 / (1 + BBSW.iloc[i - 1, 2] * BBSW.iloc[i - 1, 1]))
            )
            B_curr = 1 / (1 + BBSW.iloc[i, 2] * BBSW.iloc[i, 1])
            results.append(
                (1 / 5) * c[i] * ((ti**5) - (t**5))
                + (1 / 2) * b[i] * ((ti**2) - (t**2))
                + a[i] * (ti - t)
                + np.log(B_curr / B_prev)
            )

        def inside_instant_calculator(t):
            T = list(new_df["Term"])
            ts = [0] + list(filter(lambda x: x < t, T)) + [t]
            sum = 0
            for i in range(len(ts) - 1):
                sum += (
                    (1 / 5) * c[i] * (ts[i + 1] ** 5 - ts[i] ** 5)
                    + (1 / 2) * b[i] * (ts[i + 1] ** 2 - ts[i] ** 2)
                    + a[i] * (ts[i + 1] - ts[i])
                )
            return sum

        for i in range(len(Swap)):
            t = Swap.iloc[i, 1]
            freq = 1 / 4 if t <= 3 else 1 / 2
            payments = np.arange(freq, t, freq)
            l = Swap.iloc[i, 2] * freq
            sum_dfs = sum(
                list(map(lambda x: np.exp(-inside_instant_calculator(x)), payments))
            )
            results.append(
                inside_instant_calculator(t) + np.log((1 - l * sum_dfs) / (1 + l))
            )

        return results

    p = fsolve(solve, [100] * len(new_df) * 3)
    a = p[:points]
    b = p[points : 2 * points]
    c = p[2 * points :]
    T = list(new_df["Term"])

    def ifr_calculator(t):
        i = len(list(filter(lambda x: x < t, T)))
        return c[i] * (t**4) + b[i] * t + a[i]

    def df_calculator(t):
        ts = [0] + list(filter(lambda x: x < t, T)) + [t]
        sum = 0
        for i in range(len(ts) - 1):
            sum += (
                (1 / 5) * c[i] * (ts[i + 1] ** 5 - ts[i] ** 5)
                + (1 / 2) * b[i] * (ts[i + 1] ** 2 - ts[i] ** 2)
                + a[i] * (ts[i + 1] - ts[i])
            )
        return np.exp(-sum)

    return ifr_calculator, df_calculator

From the plot, it can be seen that the maximum smoothness instantaneous forward rate curve is very smooth without abrupt changes compared to the log-linear interpolated instantaneous forward rate curve. This is because the maximum smoothness method minimises the curvature of the instantaneous forward rate curve, resulting in a smoother and more stable curve. In contrast, the log-linear interpolation method may introduce abrupt changes in the instantaneous forward rate curve due to its piecewise linear nature. Furthermore, the maximum smoothness also tracks the overall trend of the instantaneous forward rate curve used by log-linear interpolation.

For the discount factor term structure, both methods produce very similar results, as the discount factors are derived from the instantaneous forward rates. The small differences observed are due to the different interpolation techniques used in each method.

Pricing coupon bond

Consider a coupon bond with a principal of 100 and a coupon rate of 8% per annum, where the bond matures in exactly five years and the coupons are paid quarterly.

def bond_value_calc(F, c, freq, T):
    payments = np.arange(freq, freq + T, freq)
    B1 = list(map(lambda x: df_calculator_1(x), payments))
    B2 = list(map(lambda x: df_calculator_2(x), payments))
    return (
        np.sum(np.array(B1)) * c * freq * F + B1[-1] * F,
        np.sum(np.array(B2)) * c * freq * F + B2[-1] * F,
    )
